呵呵，下面说一下如何通过 ipv6 进行翻墙。目前在Ubuntu10.04下测试成功，希望下面内容对各位有所帮助。

1.安装miredo
请在终端中，执行如下指令：
sudo apt-get install miredo

2 修改防火墙IPv6设置
以root权限打开ufw的配置文件: "/etc/default/ufw"，找到"IPV6=no"这一行，将其改为"IPV6=yes"。

3. 修改Hosts文件
完整的Hosts文件请打开http://docs.google.com/View?id=dfkdmxnt_61d9ck9ffq，将其内容粘贴至"/etc/hosts"文件末尾并保存。

4. 重启网络
sudo invoke-rc.d networking restart

完成以上步骤，在ubuntu下我们使用Ping6命令测试网络是否已经畅通,可以在终端中输入:
ping6 -c 3 youtube.com

若出现ping不通情况,首先，请检查防火墙是否允许IPv6 Teredo，在Ubuntu 10.04下，防火墙是默认禁用IPv6 Teredo的，因此会出现执行ping6指令错误的情况()。

我用的Teredo服务器地址为：teredo-debian.remlab.net
